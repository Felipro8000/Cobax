
# Componente LUI:

## Elementos
Dos Spliters 16 8 : 8
Entrada de 8 bits
Entrada de 16 bits
Salida de 16 bits

### Funcionamiento
La entrada  de 16 bits se divide con uno de los spliter y se toman los 8 bits de menor valor y estos se suman con la entrada de 8 bits y ambos se unen en el otro spliter poniendo los bits de la entrada de 16 bits como bits de mayor valor.
